## üîç What is AgentRegistry?

AgentRegistry is a security tool designed for users working with AI agents that install packages via npm, a popular package manager. It scans every package for hidden threats like malicious code, prompt injections, and supply chain attacks to prevent problems before they happen.

### Why use AgentRegistry?

- Stops harmful code before it reaches your system.
- Protects against sneaky AI prompt hacks.
- Keeps your local environment safe and clean.
- Works automatically without requiring deep technical knowledge.

## üõ† Using AgentRegistry

Once installed, AgentRegistry runs quietly in the background. Here is how it works for you:

### Adding packages to scan

When an AI agent tries to install a package, AgentRegistry intercepts the request.

- It scans the package for known security issues.
- If the package is safe, it stores it locally for quick access.
- If a threat is found, it blocks the package and notifies you.

### Managing quarantined packages

Sometimes AgentRegistry detects suspicious packages and moves them to quarantine:

- You can review quarantined packages anytime via the AgentRegistry interface.
- If you trust the package, you can release it after manual inspection.
- Otherwise, quarantined packages stay blocked to protect your system.
